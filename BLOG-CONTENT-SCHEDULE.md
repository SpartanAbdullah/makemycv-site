## How to publish each post

On the scheduled day, open the post's `.mdx` file in `content/blog/` and:

1. Change `published: false` → `published: true`.
2. (Optional) Update `date:` to the actual publish date if it differs.
3. Commit and push — Velite rebuilds on deploy and the post goes live on Vercel.

Optionally promote one pillar to the homepage Featured strip by setting
`featured: true` (only the 3 newest featured posts show, so don't exceed that).
